# IssueFlow Lite

IssueFlow Lite 是一个面向小型研发团队的轻量缺陷协作平台，也是一个用于实践 Python Web 后端开发的学习项目。

项目计划在 4 周内完成首个可用版本，重点练习从需求分析、数据库设计和接口开发，到自动化测试、GitHub CI 与公网部署的完整开发流程。

## 核心功能

- 用户注册、登录与退出
- 项目及项目成员管理
- 缺陷创建、编辑、查看和状态流转
- 缺陷负责人、严重程度和优先级管理
- 缺陷搜索、筛选、排序与分页
- 缺陷评论与操作历史
- 项目成员权限控制

## 初步技术栈

- Python 3.12+
- FastAPI
- Jinja2 + Bootstrap
- SQLAlchemy 2 + Alembic
- PostgreSQL
- pytest
- Docker Compose
- GitHub Actions
